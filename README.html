Welcome to My Personal Projects Website

This website focuses mainly on the major arduino projects that I have made throughout the years and is continuously updated as I come up with more and more ideas for different projects. All projects include a brief explanation of the project, pictures of the circuit design, a video of it working, and the code I wrote to get it to work.

About Me

I have always been interested in electronics and seeing how hardware and software integrate together to make things like a calculator or computer. This is why, in grade 10 I decided to buy an arduino kit and have been making personal projects on it since then. I have been working on personal arduino projects since the tenth grade and have been documenting my progress on this website.

Projects

Alarm Clock

The following project is an alarm clock I made using an LED display, an IR receiver, and an active buzzer.

Calculator

The following project is a calculator. This project was one of the first ones I undertook as I was interested in trying to replicate a real world device.

Color Memory Game

The following project is a memory game that requires the user to remember the sequence with which the LEDs lit up and then press the switches in that order. The number of LEDs that light up increases by one each time. This was inspired by the Simon Game.

Motion Sensor

The following project is a mechanism that is meant to detect when something is within a certain distance of it and light up two LEDs when it is close enough. It also makes a sound depending on how close it is. This is done using an HC-SR04 Ultrasonic sensor to detect the distance, as well as LEDs and an active buzzer.

Number Memory Game

This project incorporates an LED display to display randomly generated numbers that increase by 1 digit each time for a short period of time. The input of the user is read by the IR reciever and displayed on the LED screen. This was inpired by the number memory game in Human Benchmark.

Reaction Time Tester

The following project incorporates an LED display screen to set up a random value on the remote. The user then has a set amount of time to input the value. The LED screen also keeps track of their strike on the bottom display. This game was inspired by the reaction time test on human benchmark.

Automatic Container Closer

The following project uses a sound sensor and a servo to automatically close the led on my piggybank (cup) as soon as it hears a coin fall in. A switch is used to open the lid again.
